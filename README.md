# Conversational RAG with PDF

A Streamlit-based Conversational RAG application that allows users to upload PDFs
and chat with them using semantic search, embeddings, and chat history.

## Features
- PDF ingestion
- Chunking & embeddings (HuggingFace MiniLM)
- Vector store (Chroma)
- History-aware conversational retrieval
- Session-based chat memory
- Streamlit UI

## Tech Stack
- LangChain
- HuggingFace embeddings
- ChromaDB
- Groq LLMs
- Streamlit
